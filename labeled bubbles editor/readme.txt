Put pigy file from labeler in editor folder and run bat on it. Use esc to exit and save, space to switch, mouse left click and hold  to ADD sphere if required, right click to remove,  b key to apply (add/remove) selection. 

In the begining of editor file you can change parameter masht=1.7 to fit the window for your screen resolution. And at the end of file There are kor=kooor[:,2]*1.64. 

You can use this koeff=1.64 to adapt the programm for your miscroscope size/pixel number.
